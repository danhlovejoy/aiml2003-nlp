How This Course Works

Each two-week module follows the same rhythm:

Week A: Learn the Concept

  • Before Tuesday: Watch the assigned videos and do the readings (~90 min).
  • Tuesday class In Person: We discuss the concepts. I'll ask questions, you'll ask questions, and we'll make sure everyone understands the "why" before we code.
  • Thursday lab Zoom: We work through the assignment together. This is your best opportunity to get help and debug in real time.

Week B: Apply & Present

  • Before Tuesday: Finish your lab, push it to GitHub, and prepare your presentation.
  • Tuesday class In Person: You present what you built (3–5 minutes). New topic readings/videos assigned.
  • Thursday lab Zoom: We start the next lab together.
  • By Tuesday class Due: Deliverables due in GitHub (link submitted in Canvas) — see the Syllabus for exact dates.
Key point: I grade everything through your GitHub repo, including notebook contents (GitHub renders .ipynb files with all outputs). Save your final work to GitHub before the deadline. If it's not in your repo, it doesn't exist for grading.

What to Do Right Now

Work through this module in order. Each item builds on the previous one:

  1. Take the survey (next item in this module): 10 minutes, completion credit.
  2. Set up Google Colab: Follow the setup guide, run your first cell.
  3. Get your Gemini API key: Claim your free student account, create your key, store it in Colab.
  4. Create your GitHub account: Professional username, create the course repo.
  5. Connect Colab to GitHub: Save a test notebook to your repo.
  6. Set up NotebookLM: Create your first notebook and upload the Week 1 videos as sources.
  7. Watch the Week 1 videos before Tuesday's class.
Don't wait for Thursday lab to start the setup. Try the guides on your own first. Thursday lab is there to help you troubleshoot; it goes much faster if you've already attempted each step.

A Word About GenAI in This Course

I expect you to use GenAI tools in this class. It's an NLP course, after all. But remember: you are accountable for the work you submit. A computer cannot be held accountable. You'll get credit for work you do with AI, and you'll be counted off for errors GenAI tools make.

Two hard rules:

Never share your university login credentials with any external GenAI agent. Do not upload sensitive data, PII, or your school login credentials into any AI tool. Assume all inputs are public and will be used to train future models.
